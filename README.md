📊 IPL Data Analysis – Top Run Scorers
📌 Overview

This project analyzes IPL (Indian Premier League) cricket data using Python.
It focuses on identifying the top 10 run scorers based on match delivery data and visualizing the results.

📁 Dataset Used

The project uses two CSV files:

matches.csv → Contains match-level information
deliveries.csv → Contains ball-by-ball delivery data
⚙️ Technologies Used
Python 🐍
Pandas (Data Analysis)
Matplotlib (Data Visualization)
🚀 Features
Loads IPL datasets
Aggregates total runs scored by each player
Identifies Top 10 batsmen
Displays results in:
Console output
Bar chart visualization
📊 Output
Prints top 10 run scorers in descending order
Generates a bar graph showing player performance
🧠 How It Works
Load datasets using Pandas
Group data by batsman
Sum total runs (batsman_runs)
Sort values in descending order
Select top 10 players
Plot results using Matplotlib
